Organization Chart (AngularJS)

An interactive organization chart built with AngularJS 1.x, supporting drag-and-drop positions, parent-child relationships, and visual connection paths.


Features:
- Drag & Drop positions into organization levels
- Parentâ€“child hierarchy with curved connector lines
- Hover state to highlight parent and child nodes
- Delete confirmation modal with sub-item awareness
- Auto-return deleted nodes back to Hire Positions
- Permission assignment per position
- Pure AngularJS (no build step required)

Tech Stack:
- AngularJS 1.8.x
- angular-drag-and-drop-lists
- Vanilla JavaScript
- HTML / CSS
- SVG (for connector paths)

Getting Started:
No installation or build step required.
1. Clone or download this repository
2. Open `index.html` in a browser
